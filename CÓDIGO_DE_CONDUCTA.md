# Código de Conducta

Este proyecto busca proporcionar una comunidad acogedora y respetuosa para las contribuciones.

## Nuestros estándares

- Sé respetuoso y amable con todas las personas.
- Usa un lenguaje constructivo en issues, revisiones de código y discusiones.
- Trata con igual respeto a hablantes de inglés y español.
- Enfócate en el código y las ideas, no en atributos personales.

## Comportamiento esperado

- Sé considerado e inclusivo.
- Usa un lenguaje claro y educado.
- Escucha y responde de manera profesional.
- Ayuda a nuevos colaboradores a sentirse bienvenidos.

## Comportamiento inaceptable

- Acoso, amenazas o lenguaje abusivo.
- Discurso de odio, sexismo, racismo o discriminación de cualquier tipo.
- Ataques personales o comentarios insultantes.
- Interrupción deliberada de discusiones o hilos de issues.

## Informar problemas

Si experimentas o presencias un comportamiento inaceptable, informa abriendo un issue o contactando a los mantenedores a través de los canales del repositorio.

## Aplicación

Los mantenedores del proyecto pueden tomar medidas para proteger a los colaboradores y al proyecto, incluyendo eliminar contenido ofensivo, cerrar issues o pull requests abusivos y bloquear usuarios si es necesario.

## Alcance

Este Código de Conducta se aplica a todos los espacios del proyecto, incluidos el rastreador de issues, las revisiones de pull request, los mensajes de confirmación y la documentación del proyecto.
