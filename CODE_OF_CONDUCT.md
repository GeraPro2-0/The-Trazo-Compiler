# Code of Conduct

This project is dedicated to providing a welcoming and respectful community for contributors.

## Our standards

- Be respectful and kind to everyone.
- Use constructive language in issues, code reviews, and discussions.
- Treat English and Spanish speakers with equal respect.
- Focus on the code and ideas, not personal attributes.

## Expected behavior

- Be considerate and inclusive.
- Use clear, polite language.
- Listen and respond professionally.
- Help new contributors feel welcome.

## Unacceptable behavior

- Harassment, threats, or abusive language.
- Hate speech, sexism, racism, or discrimination of any kind.
- Personal attacks or insulting comments.
- Deliberate disruption of discussions or issue threads.

## Reporting

If you experience or witness unacceptable behavior, please report it by opening an issue or contacting the maintainers through the repository's communication channels.

## Enforcement

Project maintainers may take action to protect contributors and the project, including removing offensive content, closing abusive issues or pull requests, and blocking users when necessary.

## Scope

This Code of Conduct applies to all project spaces, including issue trackers, pull request reviews, commit messages, and project documentation.
